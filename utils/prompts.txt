
The attached fr.js file is a localization file for a tinymce plugin. The plugin facilitates marking up a text passage in the tinymce document for the language (it allows highlighting text, clicking on a language in the menu, and then it sets a format with a lang attribute for the language). It also allows the user to see language markup by using CSS to set different colors for the various lang attribute values found in the document, etc. 

The translation strings for the langName.xx keys are going to be listed in an alphabetically sorted dropdown (options in a select). All of the entries for a given language need to appear together. 

For proper alphabetical sorting where all variants of a language appear together, we need consistent "Language (Region/Country/Script)" formatting.
The translations need to use that consistent parenthetical format, for example: "German (Germany)", "English (Ireland)", "Chinese (Hong Kong)"

The langName translations cannot have adjective language name formats (American English, Austrian German, etc.) because if they did then all the entries for a given language would not be listed together.

The attached localization file has a few issues in the language names.
 - Issue 1: Some of the language name translations are missing. 
 - Issue 2: Some of the language names do not have the correct capitalization for the locale's culture. 
 - Issue 3: Some of the existing language translations are not in a consistent format (parenthetical format, etc.)

I have attached as PDF with screenshots of some of the menus and dialogs that the plugin presents as a reference. I have also attached the en.js localization file for a reference on the format for the language+region names.

Please update the attached fr.js localization file to fill in any missing language+region translations, and set the case of the language names as appropriate in the locale's culture (the country/region names may remain capitalized as appropriate), and of course make sure they are in the parenthetical format when not just a language code.
